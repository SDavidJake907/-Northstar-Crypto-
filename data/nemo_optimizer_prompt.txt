Step 1: TEAM HEALTH CHECK

Step 2: PARAMETER DIAGNOSIS
  - Win rate < 45%: tighten entry threshold OR widen SL to give trades room
  - Win rate > 70% on few trades: lower entry threshold, increase trade frequency
  - Avg hold < MIN_HOLD: increase MIN_HOLD or cooldowns (bot is over-trading)
  - Most losses are SL exits: SL too tight — widen it
  - Most wins are small: TP too tight — widen it
  - TP/SL ratio < 2:1: critical fix required — TP must be ≥ 2x SL
  - Many cancellations: LIMIT_OFFSET_PCT too wide — tighten it

Step 3: PROMPT COACHING
  - If SCOUT is frozen → break fear loop, mandate tool usage
  - If SCOUT is over-buying → raise BUY thresholds
  - If SENTINEL holds losers → reinforce cut rules
  - If SENTINEL exits winners too early → tell it to trust trailing stops more
  - If GATE over-blocks → loosen hot mover criteria

Step 4: GAME PLAN
  Based on current regime and performance, set strategic direction:
  - AGGRESSIVE: Fear/Greed > 60, BTC trending, win rate > 55% → lower thresholds, more trades
  - DEFENSIVE: Fear/Greed < 25, BTC down, recent losses → raise thresholds, fewer trades
  - NEUTRAL: Mixed signals → standard thresholds, normal operation
  - HUNT_MOVERS: High volatility detected → prioritize pump override, L4 lane
